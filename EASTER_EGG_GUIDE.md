# ?? Mode FUN - Easter Egg Guide

## Comment activer l'Easter Egg ?

1. Lancez l'application
2. Cliquez sur le bouton **"?? MODE FUN"** en rose dans le panneau de simulation
3. Vous verrez le mode animation s'activer

## Qu'est-ce qui se passe ?

Le mode FUN affiche une simulation amusante où :

### ?? L'Avion
- Fonce dans les **Tours Jumelles** ??

### ?? Voiture BodyPositive
- Fonce dans le **McDonald's** ??

### ?? La Petite Voiture
- S'écrase dans le **Parking** ???

### ??? La Sportive
- Fonce dans le **Mur de Briques** ??

### ?? Le Fourgon
- Traverse la **Fontaine** ?

## Fonctionnalités

? **Animation fluide** - Les véhicules se déplacent continuellement
?? **Effet d'explosion** - Lorsqu'un obstacle est touché, une animation d'explosion apparaît
?? **Visualisation colorée** - Chaque obstacle a sa propre couleur
?? **Messages amusants** - Les résultats s'affichent en texte jaune

## Retour à la simulation normale

Cliquez à nouveau sur **"?? MODE FUN"** pour revenir à la simulation normale.

## Code technique

L'easter egg est constitué de :
- **Obstacle.cs** - Classe pour les obstacles interactifs
- **VoitureEasterEgg.cs** - Véhicules spécialisés pour l'easter egg
- **EasterEggManager.cs** - Gestionnaire des obstacles
- **EasterEggController.cs** - Logique du mode FUN
- **EasterEggPanel.cs** - Panneau de visualisation avec animations

Bon amusement ! ??
